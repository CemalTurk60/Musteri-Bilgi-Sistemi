# Musteri-Bilgi-Sistemi
Görsel Programlama 2 - Müşteri Bilgi Sistemi Projesi

## Proje Hakkında
C# Windows Forms ve Microsoft SQL Server kullanılarak geliştirilmiş
masaüstü tabanlı müşteri yönetim uygulamasıdır.

## Özellikler
- Yeni müşteri ekleme
- Mevcut müşteri bilgilerini güncelleme
- Müşteri silme
- Tüm müşterileri listeleme

## Kullanılan Teknolojiler
- C# Windows Forms (.NET Framework)
- Microsoft SQL Server (LocalDB)
- Visual Studio 2022

## Veritabanı Tablosu
| Alan | Tip |
|------|-----|
| CustomerID | INT (Otomatik) |
| Name | NVARCHAR(50) |
| Surname | NVARCHAR(50) |
| Phone | NVARCHAR(20) |
| Address | NVARCHAR(255) |
| Balance | INT |

## Kullanım
1. Alanları doldur → **Ekle** ile kaydet
2. Listeden satıra tıkla → bilgiler otomatik gelir → **Güncelle**
3. Listeden seç → **Sil** → onayla
4. **Temizle** ile alanları sıfırla
